# Compilation
## XCode
```shell
git clone https://github.com/microsoft/vcpkg.git
cd vcpkg
./vcpkg install glfw3
./vcpkg install ray lib
git clone https://github.com/rodolphe74/Boulder.git
mkdir debug
cmake .. -DCMAKE_TOOLCHAIN_FILE=[vcpkg_folder]/scripts/buildsystems/vcpkg.cmake -GXcode
```
